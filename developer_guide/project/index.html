Project

The primary goal of the SnoScience project is to gain a better understanding in the workings of neural networks (NNs). A secondary goal is to get familiar with commonly used tools in Python development and to apply them in this project. The product of the project is a Python package released on PyPi capable of performing the use cases mentioned below.

Use cases

UC 1 Network setup
Pre-conditions n/a
Scenario Set up a NN of arbitrary size with customisable activation and loss functions per layer.
Post-conditions NN instance created based on the given parameters.
Exceptions 1. Activation function not supported.
2. Loss function not supported.
3. Optimiser not supported.
UC 2 Network training
Pre-conditions NN instance containing at least 1 layer.
Scenario Train a NN with an arbitrary number of samples with a customisable optimiser.
Post-conditions Neurons within the NN instance are trained.
Exceptions 1. NN does not contain any layers.
2. Inputs of samples do not match with inputs of NN.
3. Outputs of samples do not match with outputs of NN.
4. Number of samples per epoch is larger than dataset.
UC 3 Network predictions
Pre-conditions NN instance containing at least 1 layer.
Scenario Let a NN make predictions for an arbitrary number of samples.
Post-conditions NN predictions per output per sample as an array.
Exceptions 1. NN does not contain any layers.
2. Inputs of samples do not match with inputs of NN.
UC 4 Network performance
Pre-conditions Predictions per sample and true values per sample as arrays.
Scenario Calculate the performance of the NN.
Post-conditions NN performance per output as an array.
Exceptions Predictions and true values are unequal in size.

Requirements

General

The user is able to create a NN capable of being trained and making predictions:

  1. This NN can be set up with an arbitrary size:
    1. it can contain an arbitrary number of inputs.
    2. it can contain an arbitrary number of outputs.
    3. it can contain an arbitrary number of layers.
    4. each layer in this NN can contain an arbitrary number of neurons.
    5. the activation function is customizable per layer (see activation functions).
    6. the loss function is customisable per layer (see loss functions).
  2. This NN can be trained with an arbitrary number of samples:
    1. this training is customisable (see supervised learning).
    2. this training can be done using a single command.
    3. an error is raised if the inputs and outputs of the samples do not match with the NN.
    4. the optimiser is customisable (see optimisers).
    5. the selected number of samples per epoch is random and customisable.
  3. This NN can make predictions for an arbitrary number of samples:
    1. this prediction is customisable (see supervised learning).
    2. this prediction can be done using a single command.
    3. an error is raised if the inputs and outputs of the samples do not match with the NN.

Supervised learning

This NN supports the following supervised learning methods:

  1. classification.
  2. regression.
  3. an error is raised if an unsupported method is given by the user.

Activation functions

This NN supports the following activation functions:

  1. sigmoid.
  2. an error is raised if an unsupported function is given by the user.

Loss functions

This NN supports the following loss functions:

  1. mean squared error (MSE).
  2. an error is raised if an unsupported function is given by the user.

Optimisers

This NN supports the following optimisers:

  1. stochastic gradient descent (SGD).
  2. an error is raised if an unsupported optimiser is given by the user.

Metrics

This NN supports the following performance metrics:

  1. mean error and the standard deviation.
  2. accuracy.
